## 📖 About the Project

This project is a creative restaurant menu website where users can browse food categories, add items to the cart, and view the total bill.  
It also includes voice recognition, speech output, and a mouth-opening animation using camera input.

## ✨ Features

- 🎤 Voice command support
- 👄 Mouth-open gesture to start
- 🥗 Food categories with menu items
- 🛒 Add to cart and generate bill
- 💰 Live total price update
- 🔊 Speech feedback for actions
- 📷 Camera-based interaction using MediaPipe

## 🧰 Technologies Used

- HTML
- CSS
- JavaScript
- MediaPipe Face Mesh
- Web Speech API

## 🍴 Menu Categories

- 🥗 Veg
- 🍗 Non-Veg
- 🥟 Chinese
- 🥤 Drinks
- 🍦 Shakes

## 🚀 How It Works

1. Open the website.
2. Allow camera access.
3. Open your mouth or click to start.
4. Use voice commands like:
   - "Veg"
   - "Non Veg"
   - "Chinese"
   - "Drinks"
   - "Shake"
   - "Bill"
5. Add items and view your total bill.
